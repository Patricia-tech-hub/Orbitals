## Installation
1. Clone the repository:
    ```sh
    git clone <repository-url>
    cd <repository-directory>
    ```

2. Create and activate a conda environment:
    ```sh
    conda env create -f environment.yml
    conda activate project_asteroids
    ```

3. Install the package:
    ```sh
    pip install -e .
    ```
